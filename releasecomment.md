# Release Notes

Dieses Repository enthaelt ein PrestaShop-Modul, das Unterkategorien innerhalb einer Kategorie alphabetisch sortiert.

## Highlights

- Sortiert alle Unterkategorien einer gewählten Kategorie
- Kategorie kann über den Navigator gewählt werden

## Hinweise

- Details sind im README des Repos zu finden

## Installationsanleitung (PrestaShop via ZIP-Datei)

1. ZIP aus dem gewuenschten Release herunterladen
	Gehe auf die Release-Seite dieses Repositories und lade beim gewuenschten Release die Modul-ZIP herunter (Asset der jeweiligen Version).
	Verwende genau diese Release-Datei fuer die Installation in PrestaShop.

2. Im PrestaShop-Backoffice anmelden
	Oeffne dein PrestaShop-Admin (`/admin...`) mit einem Benutzer, der Module installieren darf.

3. Zu Modul-Manager wechseln
	Gehe zu `Module > Module Manager`.

4. Modul hochladen
	Klicke auf `Modul hochladen` und waehle die ZIP-Datei aus.
	PrestaShop entpackt und erkennt das Modul automatisch.

5. Modul installieren
	Wenn nicht gleich ein Konfigurieren Button erscheint: Suche nach `Internauten Categories` und klicke auf `Installieren`.

6. Modul konfigurieren
	Klicke auf `Konfigurieren`.
	Trage optional die `Parent category ID` ein oder waehle sie im Kategorie-Navigator aus.
	Speichere die Einstellungen.

7. Funktion testen
	Klicke im Modul auf `Sort now`, um die Sortierung der Unterkategorien auszufuehren.

8. Cache leeren (falls noetig)
	Wenn Aenderungen nicht sofort sichtbar sind: `Erweiterte Parameter > Performance > Cache leeren`.
